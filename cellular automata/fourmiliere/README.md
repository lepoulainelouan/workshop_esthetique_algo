###FOURMILIERE
Fourmilière est un automate cellulaire essayant de reproduire la création d'une fourmilière !!
Pour ce faire il y a différents paramètres permettant de créer cette fourmilière :

- Si la cellule vide
- Le tunnel
- La chambre
- les murs qui entourent la totalité du réseau
- Les zones spéciales comme la nurserie ou encore le stockage qui sont des réels endroits existants dans les fourmilières

Pour la création des tunnels les fourmis ici appelés "Walkers" creusent les différentes galeries.
Il n'y peut en avoir que 5 maximum afin de ne pas creuser toute la terre.
La création des tunnels préfère suivre une direction principale mais peut parfois tourner comme pour simuler la présence d'un obstacle sur le terrain
mais aussi pour créer une fourmilière plus organique et belle.

Afin de ne pas creuser toute ma terre les fourmis ont une énergie max de 6000 pas avant leur triste fin.

Les ressources et inspirations sont toutes présentes dans le dossier "reference"
